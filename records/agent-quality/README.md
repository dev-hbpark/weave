# Agent Quality Records

Evaluations of agent, skill, and routing performance for this project.

## When to write a record here

- After running `/evaluate-agents` or the `agent-quality-review` skill.
- After a weekly review (`/weekly-review`) flags an underperforming agent or skill.
- After an incident postmortem identifies a routing or agent failure as a contributing factor.
- After a routing-table change to record the rationale.

## Naming

```
AGENT_EVALUATION-<NNN>-<short-slug>.md
```

Examples:

- `AGENT_EVALUATION-001-w20-snapshot.md`
- `AGENT_EVALUATION-002-decision-router-tuning.md`

## Template

Use `AGENT_EVALUATION_TEMPLATE.md` as the starting body.
